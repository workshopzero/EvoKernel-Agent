**ECK (Evo-Cognition Kernel)** is a geek-oriented, "life-like" AI Agent infrastructure. Instead of relying on heavy frameworks like LangChain, it builds a closed-loop cognitive system from scratch, featuring **Dynamic Routing, Subconscious Dreaming, AST Physical Sandboxing, Schema Memory, and even Self-Modification of its own kernel**.

## 🌟 Core Features
- 🧠 **4D Memory System**: STM (Context), Knowledge, Genome (Tools), and Schema (Muscle Memory).
- 💤 **Subconscious Dreaming Engine**: Autonomously invents new Python tools in the background when the PC is idle.
- 🛡️ **AST Sandbox & Anomaly Detection**: A secure execution sandbox with memory leak monitoring and infinite-loop detection.
- 🔁 **Dynamic Failover Routing**: Seamlessly switches between Cloud (Google Vertex AI) and Local (Ollama) models.
- 🧬 **Auto-Reflection**: Analyzes tracebacks to fix its own bugs automatically.

## 🚀 Quick Start
```bash
git clone https://github.com/workshopzero/EvoKernel-Agent.git
cd EvoKernel-Agent
pip install -r requirements.txt

Rename .env.example to .env and fill in your configs.

# Start the Watchdog daemon
python watchdog.py

🕹️ CLI Commands

    /set_mode <auto|local|cloud|offline> : Force execution mode.
    /toggle_human <on|off> : Toggle human confirmation lock.
    /toggle_reflect <on|off> : Toggle auto-reflection and bug fixing.
    /review : Review mutated skills generated by the subconscious.
